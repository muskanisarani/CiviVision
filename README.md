# CiviVision 🌍

CiviVision is a civic issue reporting and monitoring platform developed as part of an international hackathon.

## Features
- AI-powered civic complaint reporting
- User dashboard with complaint tracking
- Admin dashboard for issue monitoring
- Clean, responsive UI
- Map visualization (demo placeholder)

## Tech Stack
- HTML
- CSS
- JavaScript
- Bootstrap
